Detailed Description

template<typename T, std::size_t N, std::size_t ... O>
class mdc::DArray< T, N, O >

Represent an array with a fixed size for each dimension.

Template Parameters
TType of the elements stored
NSize of the outer-most dimension
OParameter pack of the following sizes
